What is Remotly?

Remotly is an all-in-one virtual office for remote teams — a virtual office floor, real-time chat, video & audio calls, Kanban project management, and automatic time tracking, all in one browser tab. This connector keeps Remotly's projects and tasks in perfect sync with Odoo.

What it syncs.

Map once, then forget about it.

📁

Projects

Odoo projects become Remotly projects with a board and columns, created automatically.

✅

Tasks

Tasks sync with title, description, priority and due date — created and updated both ways.

📋

Stages ↔ Columns

Odoo stages map to Remotly board columns by name; missing columns are created on the fly.

👤

Users & assignees

Odoo users are linked to Remotly users by email and set as task assignees.

🔄

Two-way

Push on change (real-time) and pull on a schedule. A loop guard prevents echo updates.

🔒

Secure API key

Connects with a Remotly API key — no password stored. Multi-company aware.

See it in Odoo.

Configure once, sync forever.

The Remotly backend — paste the URL + API key, pick the sync direction, and you're connected.

Enable "Sync with Remotly" on any project — its tasks flow to a Remotly board.

Run a manual two-way sync any time from the wizard, or let the schedule do it.

Up and running in 3 steps.

1

Generate an API key

In Remotly → Company settings → API keys.

2

Add the backend in Odoo

Paste the URL + key, click Test connection.

3

Enable sync on projects

Tick "Sync with Remotly" — done.

Requires a Remotly account. Works on Odoo 18 & 19.
