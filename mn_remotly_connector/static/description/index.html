What is Remotly?

Remotly is an all-in-one virtual office for remote teams — one browser tab that replaces Zoom, Slack, Asana and Toggl. A live office floor, chat, video, Kanban project management, time tracking and analytics, together. Built with full Arabic / RTL support for MENA and global teams. remotlyapp.com

The Remotly backend in Odoo — connected and syncing two-way.

Enable "Sync with Remotly" on any Odoo project.

Run a manual two-way sync, or let the schedule do it.

Up and running in 3 steps.

1

Generate an API key

In Remotly → Company settings → API keys.

2

Add the backend in Odoo

Paste the URL + key, click Test connection.

3

Enable sync on projects

Tick "Sync with Remotly" — done.

Requires a Remotly account (remotlyapp.com). Works on Odoo 18 & 19.
